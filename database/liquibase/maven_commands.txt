To clean and install including the test data locally on postgres you can run:

  mvn clean test -Ppostgres,local,testdata

for an oracle install use

  mvn clean test -Poracle,maredev,testdata

See https://webgate.ec.europa.eu/CITnet/confluence/display/UNIONVMS/Use%2C+configure+and+install+db-liquibase+project
